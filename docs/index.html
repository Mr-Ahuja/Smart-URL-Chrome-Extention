Copy URL Smartly – CURLs

Support, help, and contact for the CURLs Chrome extension. Quickly copy a clean, labeled link to your clipboard in HTML and plain text.

What CURLs Does

  • One‑click copy of active tab as HTML + plain text
  • Auto‑copy on popup open
  • Custom title with visual copy feedback
  • Privacy‑friendly: 100% local, minimal permissions

Get Help

  • Install unpacked via chrome://extensions
  • Reload to pick updates; check “Errors” if anything fails
  • Open an issue on GitHub for bug reports
